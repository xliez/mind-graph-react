# mind-graph-react

简单的思维导图渲染实现

## Demo

### Live Demo

https://xliez.github.io/mind-graph-react/

### Local Demo

```shell
npm run dev
```

## Features

- [x] 自动布局
- [x] 节点完全自定义
- [x] 直线 & 贝塞尔曲线
- [x] 支持拖拽
- [x] 支持缩放
- [x] 折叠节点


## Technology

- React
- Typescript
- D3.js
- [自动布局算法](https://zhuanlan.zhihu.com/p/573659320)基础上做了方向转换
